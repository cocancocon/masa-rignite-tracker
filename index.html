How teammates use this

1. Everyone opens the same GitHub Pages link.
2. Everyone enters the same Supabase URL, anon key, and room ID.
3. First person clicks Push now once.
4. Others click Pull latest once.
5. After that, changes should appear live without refresh.

Important: Do not paste private passwords or sensitive personal data here. Anyone with the site, anon key, and room ID can edit this tracker.
