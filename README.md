# MASA R-Ignite Supabase Team Tracker

This folder is ready for GitHub Pages hosting.

## GitHub Pages hosting
1. Create a new GitHub repository.
2. Upload all files in this folder to the repository root.
3. Keep the tracker file named `index.html`.
4. Go to **Settings → Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select branch `main` and folder `/root`.
7. Save, wait for the Pages URL, then share it with your team.

## Shared editing
The tracker saves locally by default. To make it shared, create the Supabase table using `supabase-setup.md`, then use the Sync tab in the tracker.
